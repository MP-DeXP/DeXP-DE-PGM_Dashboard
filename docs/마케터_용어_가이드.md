# 마케터 용어 가이드

## 목적
이 문서는 APF 대시보드의 내부 약어 중심 용어를 마케터/MD 실무 언어로 해석하기 위한 안내서입니다.
기존 기술 문서(`docs` 내 기존 파일)는 그대로 두고, 운영/실행 관점에서 바로 읽을 수 있게 정리했습니다.

## 1) 용어 사전
- `AA` -> `첫구매 유입상품`
- `PCA` -> `재구매 시작상품`
- `CA` -> `장바구니 확장상품`
- `BHI` -> `브랜드 구조 건강도`
- `BII` -> `브랜드 실전 체력`
- `BII 90/365` -> `90일 체력 대비 연간 체력`
- `Transition` -> `전환 흐름`
- `Journey` -> `고객 흐름`
- `Brand Fitness` -> `브랜드 체력 현황`

## 2) 유형 라벨(화면 표기)
### 첫구매 유입상품 유형
- `BROAD` -> `첫구매 많음`
- `QUALIFIED` -> `재구매 가능성 높음`
- `HEAVY` -> `고객 가치 높음`

### 재구매 시작상품 유형
- `CORE` -> `단골의 시작점`
- `DEEP` -> `계속 찾는 상품`
- `SCALE` -> `효자 상품`

### 장바구니 확장 유형
- `CORE` -> `장바구니 중심형`
- `PAIR` -> `함께 담는 조합형`
- `SET` -> `세트 확장형`
- `NONE` -> `신호 없음`

## 3) 인사이트 화면 읽는 법
### 인사이트 요약
- 브랜드 실전 체력과 90일 대비 연간 추세를 먼저 확인합니다.
- 경고 칩은 즉시 확인이 필요한 리스크 신호입니다.

### 첫구매 고객 흐름
- 첫구매 고객이 7/30/90일에 어떻게 재구매로 이어지는지 봅니다.
- `재구매 시작까지 평균 일수`가 길어지면 CRM 시점 조정이 필요합니다.

### 재구매 시작 전환 흐름
- 어떤 유입상품이 어떤 재구매 시작상품으로 이어지는지 확인합니다.
- 상위 경로 집중도가 높으면 대체 경로 실험이 필요합니다.

### 장바구니 확장 인사이트
- 동반구매 비율, 장바구니 크기, 유형 분포로 확장 구조를 봅니다.
- 조합형 집중도가 과하면 특정 조합 의존 리스크를 점검합니다.

### 브랜드 체력 현황
- 7/30/90/365일 체력 흐름과 추세를 함께 확인합니다.
- 하단 `브랜드 구조 건강도`는 참고값이며, 메인 운영 지표는 `브랜드 실전 체력`입니다.

### 실행 카드
- `근거 지표` -> `권장 액션` -> `예상 영향` 순으로 읽습니다.
- 마케팅/MD 실행안을 분리해 우선순위(P1/P2/P3)로 운영합니다.

## 4) 자주 헷갈리는 표현
- `브랜드 구조 건강도`는 구조의 균형(체질)이고,
  `브랜드 실전 체력`은 그 구조가 실제 기간 성과로 발현된 상태입니다.
- `첫구매 유입상품`은 유입을 만들고,
  `재구매 시작상품`은 단골 사슬의 시작을 만듭니다.
- `장바구니 확장상품`은 같은 주문 안에서 함께 담기도록 확장을 돕는 역할입니다.

## 5) 호환성 안내
- 화면 용어는 한글 중심으로 표시됩니다.
- CSV 파일명/컬럼명/코드 내부 키는 기존 기술 명칭을 유지합니다.
