# 마케터 용어 가이드

이 화면은 내부 모델 용어 대신 실무에서 바로 읽히는 표현만 사용합니다.

## 1) 화면에서 쓰는 쉬운 용어
- 첫구매 유입 상품: 신규 고객 유입을 만드는 상품
- 재구매 연결 상품: 다음 구매로 이어지게 만드는 상품
- 장바구니 확장 상품: 함께 담기며 객단가를 키우는 상품
- 브랜드 구조 건강도: 상품 구조의 균형 상태
- 브랜드 실전 건강도: 실제 매출/재구매 성과 건강도

## 2) 4분면 읽는 법
- X축: 첫구매 유입
- Y축: 재구매 연결
- 집중뷰: 점이 몰릴 때 읽기 쉽게 확대한 보기
- 원본 보기: 전체 분포를 그대로 보는 보기

## 3) 상품명이 길 때
- 표에서는 2줄만 보여줍니다.
- 상품명을 누르면 전체 이름과 상품 ID를 크게 확인할 수 있습니다.

## 4) 설정 메뉴
사이드바 하단 `설정`에서 관리합니다.
- 상품 그룹: 3단계 마법사(대상 선택 → 작업 선택 → 검토/저장)로 같은 상품의 다른 ID를 묶거나 해제
- 데이터 관리: CSV 업로드, 로컬 다시 불러오기, 저장 데이터 초기화

## 5) 하드룰
- UI/툴팁은 해요체를 사용합니다.
- UI/툴팁에는 내부 약어(AA/PCA/CA/BHI/BII/PGM)를 노출하지 않습니다.
