# 마케터 용어 가이드

## 목적
이 문서는 APF 대시보드의 내부 약어 중심 용어를 마케터/MD 실무 언어로 해석하기 위한 안내서입니다.
기존 기술 문서(`docs` 내 기존 파일)는 그대로 두고, 운영/실행 관점에서 바로 읽을 수 있게 정리했습니다.

## 1) 용어 사전
- `AA` -> `첫구매 유입상품`
- `PCA` -> `재구매 시작상품`
- `CA` -> `장바구니 확장상품`
- `BHI` -> `브랜드 구조 건강도`
- `BII` -> `브랜드 실전 체력`
- `BII 90/365` -> `90일 체력 대비 연간 체력`
- `Transition` -> `전환 흐름`
- `Journey` -> `고객 흐름`
- `Brand Fitness` -> `브랜드 체력 현황`

## 2) 유형 라벨(화면 표기)
### 첫구매 유입상품 유형
- `BROAD` -> `첫구매 많음`
- `QUALIFIED` -> `재구매 가능성 높음`
- `HEAVY` -> `고가치 고객`

### 재구매 시작상품 유형
- `CORE` -> `단골의 시작점`
- `DEEP` -> `계속 찾는 상품`
- `SCALE` -> `매출 확장형 상품`

### 장바구니 확장 유형
- `CORE` -> `장바구니 중심축`
- `PAIR` -> `결합형 동반상품`
- `SET` -> `세트 확장상품`
- `NONE` -> `미분류`

## 3) 인사이트 화면 읽는 법
### 인사이트 요약
- 브랜드 실전 체력과 90일 대비 연간 추세를 먼저 확인합니다.
- 경고 칩은 즉시 확인이 필요한 리스크 신호입니다.

### 첫구매 고객 흐름
- 첫구매 고객이 7/30/90일에 어떻게 재구매로 이어지는지 봅니다.
- `재구매 시작까지 평균 일수`가 길어지면 CRM 시점 조정이 필요합니다.

### 재구매 시작 전환 흐름
- 어떤 유입상품이 어떤 재구매 시작상품으로 이어지는지 확인합니다.
- 상위 경로 집중도가 높으면 대체 경로 실험이 필요합니다.

### 장바구니 확장 인사이트
- 동반구매 비율, 장바구니 크기, 유형 분포로 확장 구조를 봅니다.
- 조합형 집중도가 과하면 특정 조합 의존 리스크를 점검합니다.

### 브랜드 체력 현황
- 인사이트 필터 아래에서 먼저 보이는 `브랜드 체력 현황` 요약 카드로 현재 방향을 먼저 판단합니다.
- 핵심 읽기 순서: `체력 방향` -> `최근 기준 체력` -> `90일 대비 연간 흐름` -> `신뢰도`.
- `원인 자세히 보기`를 펼치면 구성 요소, 추세 차트, 기간별 수치를 함께 확인할 수 있습니다.
- 상세 구성 요소는 `브랜드 구조 건강도`, `매출 기여`, `재구매 강도`로 표기됩니다.
- 브랜드 구조 건강도의 3개 구조(`유입 구조 균형`, `재구매 연결 균형`, `가치 준비도`)는 요약 영역에서 바로 확인할 수 있습니다.
- 3개 구조는 레이더 차트로도 함께 표시되어 약한 축을 빠르게 확인할 수 있습니다.
- `가치 준비도 영향 요소` 카드에서 `효율·고가치 유입 비중`, `확장형 유입 비중`, `유입 집중도`, `재구매 연결 균형`을 함께 확인합니다.
- 메인 운영 지표는 `브랜드 실전 체력`이며, `브랜드 구조 건강도`는 원인 해석용 참고 지표입니다.

### 실행 카드
- `근거 지표` -> `권장 액션` -> `예상 영향` 순으로 읽습니다.
- 마케팅/MD 실행안을 분리해 우선순위(P1/P2/P3)로 운영합니다.

## 4) 자주 헷갈리는 표현
- `브랜드 구조 건강도`는 구조의 균형(체질)이고,
  `브랜드 실전 체력`은 그 구조가 실제 기간 성과로 발현된 상태입니다.
- `첫구매 유입상품`은 유입을 만들고,
  `재구매 시작상품`은 단골 사슬의 시작을 만듭니다.
- `장바구니 확장상품`은 같은 주문 안에서 함께 담기도록 확장을 돕는 역할입니다.

## 5) 호환성 안내
- 화면 용어는 한글 중심으로 표시됩니다.
- CSV 파일명/컬럼명/코드 내부 키는 기존 기술 명칭을 유지합니다.
